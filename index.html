iBabble

iBabble is a chat app built with Flask and a front-end stack of HTML, CSS, JavaScript for a user-friendly interface. Flask allows for efficient handling of connections and operations such as authentication.

Password Manager

The Password Manager, built using Tkinter in Python, enables users to save and easily access their passwords at any time. To enhance security, the program is designed to initially appear as a calculator, providing an added layer of protection for the user's stored passwords.

Google Clone Home Page

A replica of the Google homepage, built using HTML and CSS. This project aims to replicate the layout, design and functionality of the original page, demonstrating my skills in web development and front-end design.

Digital Clock

A digital clock created using HTML, CSS, and JavaScript. The clock features a clean, modern design and accurately displays the current time. The project showcases my skills in web development, specifically in the use of HTML, CSS, and JavaScript to create dynamic and interactive user interfaces.

Snake Game

A Snake game developed using the Pygame library in Python. The game features a classic gameplay, colorful graphics and smooth movement. The project exhibits the capabilities of Pygame library in creating interactive and engaging games.

Tic Tac Toe

A Tic-Tac-Toe game built using the Tkinter library in Python. The game features a user-friendly interface, the ability for players to play against each other or against the computer, and a clean design. It showcases the ability to design and develop a functional game using the Tkinter library in Python.
